📖 电池电压解析说明

ID 0x51 - 电池总电压

  • 数据格式：MOTOROLA (大端序)
  • 起始位：8 (从第0-1字节提取)
  • 数据长度：2字节 (16位)
  • 单位：V (伏特)
  • 精度：0.1V

计算公式： 电压 = (字节0 << 8 | 字节1) × 0.1

示例：

CAN帧: 51#001234567890ABCD
解析: 字节0=0x00, 字节1=0x12
原始值: 0x0012 = 18
电压: 18 × 0.1 = 1.8V
